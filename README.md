# Data Cleaning & Reporting Automation

This project automates data cleaning and reporting workflows using Python and Power BI.

## Project Objectives
- Handle missing values automatically
- Remove duplicate records
- Standardize inconsistent data
- Generate automated reports and visual summaries
- Improve data preprocessing efficiency

## Technologies Used
- Python
- Pandas
- NumPy
- Power BI
- Matplotlib
- Seaborn

## Automation Features
- Missing value handling
- Duplicate removal
- Data preprocessing workflow
- Automated report generation
- KPI reporting dashboard

## Dashboard Features
- Total Records KPI
- Missing Values Removed KPI
- Duplicate Records Removed KPI
- Ticket Status Analysis
- Ticket Type Distribution
- Priority Analysis
- Trend Analysis

## Key Insights
- Automated preprocessing improved data consistency.
- Duplicate records were removed successfully.
- Missing values were handled efficiently.
- Reporting workflow became faster and more organized.
